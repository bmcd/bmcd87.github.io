    • Toktogi

    A popup Korean-English dictionary extension

    • Available on Chrome and Firefox
    • Allows users to quickly get the English definition of a Korean word

    Based on Open Source dictionary

    • Uses 'kengdic', an open source Korean-English dictionary developed by Joe Spiegle
    • Eventually, Toktogi users will be able to directly contribute corrections, new entries, and more back to 'kengdic' to benefit all users of the dictionary

    • JQuery
    • Rails (website)

    Puzzle Fight (Source)

    A competitive online puzzle game built with Node.js

    • Created in JavaScript using the HTML5 Canvas.
    • The back-end is a Node.js server running on Heroku.

    Gameplay

    • Similar to tetris, the player drops blocks into a grid and tries to make them disappear, which sends them to the other player.
    • Land a circle adjacent to a block of the same color to destroy them, create combos to multiply the number of blocks sent over to your opponent.

    Technologies Used

    • JavaScript
    • Node.js
    • JQuery
    • HTML5
    • CSS
    • Heroku
