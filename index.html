imgDropper

imgDropper is a Rails based web app that allows the user to easily upload and share images. I wanted to recreate a popular website (Imgur) to learn new things and pull together everything I have learned recently into one complete application. Building a dynamic image sharing site from the ground up using Rails, CSS, and jQuery was a great learning experience.
